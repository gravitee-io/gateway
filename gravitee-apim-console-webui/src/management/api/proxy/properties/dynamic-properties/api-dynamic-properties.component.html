API dynamic properties

Dynamic properties let you fetch API properties from a remote resource, such as web service. You can set a polling frequency property and apply response transformation thanks to the JOLT framework . Like classical properties, dynamic properties will be injected into the expression language context. Set-up guide
1. Configure Scheduler with a cron expression to define the polling frequency.
2. Set the HTTP endpoint to fetch the properties.
